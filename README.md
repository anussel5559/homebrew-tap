# Anussel5559 Tap

## How do I install these formulae?

`brew install anussel5559/tap/<formula>`

Or `brew tap anussel5559/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "anussel5559/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
